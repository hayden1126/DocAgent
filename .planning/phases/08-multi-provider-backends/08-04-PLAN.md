<objective>
Wire the CLI surface and the orchestrator integration. Three deltas:
(1) `--backend {agent_sdk,litellm}` flag on `docagent init` and
`docagent update`, defaulting to `agent_sdk`; (2)
`BudgetTracker.add(..., external_cost: float | None = None)` accepts a
pre-computed per-call cost so the LiteLLM path can substitute the
shim's value for `pricing.estimate_cost()`'s; (3) orchestrator passes
`response.cost_usd` as `external_cost` through BOTH existing
`tracker.add()` call sites — the per-task attribution at line ~223
AND the `plan()`-call drainage loop at line ~166.

The two-call-site point is load-bearing: Phase 6's `how_to_guides`
artifact makes one LLM call inside `plan()` (topic discovery) and
N additional calls in `generate()` (one per topic). The Phase 6 P0
fix landed the drainage loop at line ~166 to attribute the plan-call
TOKENS correctly; this plan does the symmetric COST-attribution work
on the new LiteLLM seam. Missing one site = silently dropped cost on
multi-task LiteLLM artifacts, AND spurious Phase-5 `_warned_models`
WARNs because `pricing.estimate_cost()` falls back to the Opus rate
table for a Gemini/OpenRouter model string.

Purpose: After this plan, `docagent init --backend litellm --model
gemini/gemini-2.5-pro` runs end-to-end on a user's machine (with
`GEMINI_API_KEY` set), `--max-cost $X` enforces a soft cap across
both backends via the same `BudgetTracker.would_exceed()` check that
ships in Phase 5 — no orchestrator-level branching — AND multi-task
artifacts attribute every per-call cost correctly under either
backend.

Output: The user-visible feature lands. Plans 08-05 and 08-06 add
deeper integration tests + a snapshot + the new how-to doc surfacing
the feature; the actual feature works after this plan.

BudgetTracker.add() new signature (additive, backward-compatible):

```python
def add(
    self,
    model: str | None,
    input_tokens: int,
    output_tokens: int,
    tool_calls: int,
    external_cost: float | None = None,
) -> float:
    if external_cost is not None:
        per_call = external_cost
    else:
        per_call = estimate_cost(model, input_tokens, output_tokens)
    self.input_tokens += input_tokens
    self.output_tokens += output_tokens
    self.tool_calls += tool_calls
    self._cost += per_call
    return per_call
```
